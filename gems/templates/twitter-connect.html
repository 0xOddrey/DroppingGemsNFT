How It Works JavaScript for Beginners

Using a Twitter Api and NLP Sentiment Analysis, each tweet is a assigned two numbers: one for polarity and one for subjectivity. Polarity is a range from Negative emotions like Dislike, Frustration, Sadness, Anger to Positive emotions like joy, happiness, appreciation. Subjectivity is on a range from very subjective ie opinions to very objective ie factual. We use these two number to act as coordinates. Using the coordinates, we identify one pixel on a color wheel an that color represents that intersection between subjectivity and polarity. These means the color combination for each Gem is inifinite.
